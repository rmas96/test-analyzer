# Automated Test Result Analyzer

Python-based tool to analyze automated test results and generate concise reports.
Focused on data validation, result classification, and reporting in a QA context.

## Features
- Load test results from CSV or JSON
- Validate required fields
- Classify test outcomes (PASS / FAIL / ERROR)
- Generate summary reports

## Usage
```bash
python src/main.py --input data/sample_results.csv
